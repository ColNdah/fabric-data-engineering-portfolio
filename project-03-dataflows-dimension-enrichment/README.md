# Dataflows Gen2 Dimension Enrichment (Microsoft Fabric)

## 📌 Overview
This project demonstrates how to use **Dataflows Gen2** to ingest, clean, and enrich dimension data using:

- Power Query (M)
- Dataflows Gen2
- Fabric Lakehouse Delta tables

It shows how low-code transformations integrate with code-first Lakehouse engineering.

## 📁 Project Structure



## 🔧 How It Works

### **1. Ingestion**
Dataflows Gen2 ingests raw dimension data from CSV/API/SQL sources.

### **2. Transformation**
Power Query applies:
- Type conversions  
- Standardisation  
- Cleansing  
- Business rules  

### **3. Output**
Data is written to:
- Silver Delta tables  
- Gold curated dimension tables
