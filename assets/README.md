# Microsoft Fabric Data Engineering Portfolio

This repository showcases a collection of hands-on **Microsoft Fabric** data engineering projects demonstrating practical skills in:

- Fabric Lakehouse
- Delta Lake
- PySpark
- SQL
- Fabric Pipelines
- Dataflows Gen2
- Medallion Architecture (Bronze → Silver → Gold)
- ELT orchestration
- Data modelling and curation

These projects are designed to reflect real-world data engineering patterns used in modern Azure/Fabric environments.
